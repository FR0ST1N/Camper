URL Shortener Microservice

Example creation usage:

https://url-shortener-njs.glitch.me/new/https://www.google.com

https://url-shortener-njs.glitch.me/new/http://foo.com:80

Example creation output

{ "original_url":"https://www.google.com", "short_url":"https://url-shortener-njs.glitch.me/5439" }

Usage:

https://url-shortener-njs.glitch.me/5439

Will redirect to:

https://www.google.com
